# Neuro UI Judge

## Mentions

### 2026-04-26 — fMRI-based reward function for UI design
<!-- source_hash: dff89411b86d95696a760b2341513ae71c1279ce4d158f02104e1f636a1da7ec -->
Neuro UI Judge ingests HTML or CSS — whether human-authored or agent-generated — renders it, and scores it using a neuroscience model trained on fMRI data showing which brain regions activate in response to visual media. The key insight was that UI design preference is inherently subjective and poorly served by current LLM objective functions; simulating brain activation allows creation of a reward function that can drive reinforcement learning to improve AI-generated designs. The team noted that AI-designed UIs are currently easy to identify as such, which they attributed largely to a data scarcity problem.
