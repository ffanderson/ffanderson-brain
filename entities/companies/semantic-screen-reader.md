### 2026-04-26 — LLM-based accessibility tool for blind users
<!-- source_hash: dff89411b86d95696a760b2341513ae71c1279ce4d158f02104e1f636a1da7ec -->
Dylan and his partner Jean, both MIT Visualization Group PhD students, presented Semantic Screen Reader, which uses an LLM to prune and semantically relabel the complex node tree that standard screen readers expose. Instead of nodes named 'region' or 'non-client view,' users see labels like 'workspace' or 'channel.' One of the presenters appeared to be a low-vision user himself, lending credibility to the use case.
