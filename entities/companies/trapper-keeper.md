# Trapper Keeper

## Mentions

### 2026-04-26 — LLM-based focus and distraction intervention agent
<!-- source_hash: dff89411b86d95696a760b2341513ae71c1279ce4d158f02104e1f636a1da7ec -->
Trapper Keeper is a focus agent that collects keystroke and browsing data, then uses an LLM every 15 seconds to judge whether the user is on task. Rather than hard site-blocking rules, it learns from user feedback and applies an escalating set of interventions — from a pop-up nudge to locking the computer entirely — adapting based on user reactions. The team noted the agent surprised them multiple times by proactively closing distracting apps while they were working with it.
